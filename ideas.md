# デザインブレインストーミング

## 前提
税理士法人向けBtoBコーポレートサイト。ブランド印象は「誠実・知的・実務に強い・伴走型・数字と再現性」。参考サイト(lany.co.jp)のトーン&マナーを参考にしつつ、独自のデザインを構築する。

## Idea 1: "Swiss Precision" — スイスタイポグラフィ × ジャパニーズミニマリズム

### Design Movement
スイスデザイン（International Typographic Style）と日本の「間」の美学を融合。情報の正確性と余白の美しさで、税理士法人の「知的で誠実」な印象を表現する。

### Core Principles
1. **グリッドの厳密さ:** 8pxベースのグリッドシステムで、すべての要素を数学的に配置
2. **タイポグラフィが主役:** 大胆なフォントサイズのコントラストで視覚的ヒエラルキーを構築
3. **引き算の美学:** 装飾を極限まで削ぎ落とし、コンテンツそのものを際立たせる
4. **モノトーン + 1アクセント:** 白・黒・グレーの階調に、深い藍色（#1e3a5f）を一点投入

### Color Philosophy
白と黒のコントラストは「明確さ」を、藍色は「深い知性」を表現。グレーの階調は「段階的な信頼構築」を暗示する。

### Layout Paradigm
非対称グリッドレイアウト。左側に大きなタイポグラフィ、右側にコンテンツという分割構成。セクション間は大きな余白で呼吸させる。

### Signature Elements
1. 英語ラベルの大きなセクション見出し（薄いグレーで背景的に配置）
2. 細い水平線（1px）によるセクション区切り
3. 数字を大きく表示する「成果の可視化」パターン

### Interaction Philosophy
控えめで上品なインタラクション。ホバー時のテキストカラー変化、スクロール連動のフェードイン。派手なアニメーションは排除。

### Animation
スクロールに連動した要素のフェードアップ（translateY: 20px → 0, opacity: 0 → 1）。トランジションは300ms ease-out。カードホバー時の微細なtranslateY(-4px)。

### Typography System
- 見出し: Noto Sans JP (700) — 力強さと可読性
- 本文: Noto Sans JP (400) — 安定した読みやすさ
- 英語ラベル: Inter (600) — モダンで洗練された印象

## Idea 2: "Fluid Trust" — 有機的曲線 × データドリブンの信頼感

### Design Movement
Organic Modernism（有機的モダニズム）。自然界の曲線と、データの正確性を融合。「伴走型」「再現性」というブランド価値を、流れるような曲線と明確な数値表現で体現する。

### Core Principles
1. **流動的な曲線:** セクション背景に緩やかなウェーブやブロブ形状を配置し、硬くなりがちなBtoBサイトに柔らかさを加える
2. **データの可視化:** 数字・成果を大きく、グラフィカルに表示し、「再現性」を視覚的に証明
3. **グラデーションの深み:** 単色ではなく、微細なグラデーションで奥行きと上質感を演出
4. **カードの浮遊感:** シャドウとボーダーラディウスで、情報が空間に浮かんでいるような表現

### Color Philosophy
深いネイビー（#0f172a）をベースに、スカイブルー（#38bdf8）からインディゴ（#4f46e5）へのグラデーションをアクセントに。白い背景との対比で「信頼」と「先進性」を同時に表現。

### Layout Paradigm
フルワイドセクションの積み重ね。各セクションが独自の背景処理（曲線、グラデーション、パターン）を持ち、スクロールするたびに新しい世界が展開される。

### Signature Elements
1. SVGによる有機的な波形のセクションディバイダー
2. 大きな数字（成果指標）+ 小さなラベルの組み合わせ
3. グラスモーフィズム風のカード（backdrop-blur + 半透明背景）

### Interaction Philosophy
流れるようなスムーズさ。要素のスタガーアニメーション、パララックス効果、ホバー時のスケール変化。ユーザーの操作に対して「応答している」感覚を重視。

### Animation
Intersection Observerによるスクロール連動アニメーション。各要素が下から上へ、左から右へと順番にフェードイン（stagger: 100ms間隔）。CTAボタンのパルスアニメーション。

### Typography System
- 見出し: Noto Sans JP (900) — 極太で力強いインパクト
- 本文: Noto Sans JP (400) — 読みやすさ重視
- 英語ラベル: Outfit (500) — ジオメトリックで現代的

## Idea 3: "Editorial Authority" — エディトリアルデザイン × プロフェッショナルの権威性

### Design Movement
エディトリアルデザイン（雑誌・書籍のレイアウト手法）をWebに適用。高品質なビジネス誌のような「読ませる」デザインで、専門家としての権威性と信頼感を構築する。

### Core Principles
1. **コラム構造:** 雑誌のような2〜3カラムレイアウトで、情報を整理しながらも視覚的な変化を生む
2. **タイポグラフィの劇的なコントラスト:** 極端に大きな見出しと、整然とした本文のサイズ差で視線を誘導
3. **写真・ビジュアルの大胆な使い方:** フルブリード画像やオーバーラップする要素で、ダイナミックな構成を実現
4. **セリフ × サンセリフの混在:** 見出しにセリフ体を使い、本文はサンセリフ体。知的で格調高い印象を演出

### Color Philosophy
オフホワイト（#fafaf9）をベースに、ダークネイビー（#1a1a2e）をテキストに。アクセントはバーガンディ（#7c2d12）で、落ち着いた権威性を表現。「派手さ」ではなく「格」で勝負する配色。

### Layout Paradigm
非対称の2カラム構成を基本とし、テキストブロックとビジュアルブロックが交互に配置される。時折フルワイドのブレイクアウトセクションを挟み、リズムを生む。

### Signature Elements
1. 大きなドロップキャップ（段落の先頭文字を大きく表示）
2. 細い罫線と大きな余白による「紙面」感
3. プルクォート（引用文を大きく抜き出して表示）

### Interaction Philosophy
「ページをめくる」ような体験。スクロールに連動した要素の出現は、本のページが開くようなイメージ。過度なアニメーションは避け、コンテンツの読みやすさを最優先。

### Animation
テキストのフェードイン（opacity: 0 → 1, 500ms ease）。画像のクリップパス展開アニメーション。スクロール進行に応じたプログレスバー。

### Typography System
- 見出し: Noto Serif JP (700) — 格調高く、権威的
- 本文: Noto Sans JP (400) — 可読性重視
- 英語ラベル: DM Sans (500) — クリーンでモダン
